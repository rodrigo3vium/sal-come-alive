

## Studio SAL — Site Recreation Plan

### Overview
Recreate the Studio SAL personal branding agency website — a sophisticated, elegant single-page site with smooth scrolling sections, earthy/nature-inspired color palette (dark greens, warm beiges), and serif typography.

### Color Palette & Design
- **Primary colors**: Deep forest green (#2D3A2D), warm beige/cream (#F5F0EB), off-white
- **Typography**: Elegant serif font (e.g., Playfair Display) for headings, clean sans-serif for body text
- **Style**: Minimalist, luxurious, organic feel with generous whitespace

### Sections to Build

1. **Navigation Bar**
   - Logo "STUDIO SAL" on the left
   - Links: início, sobre nós, serviços (with dropdown)
   - CTA button "QUERO SALTAR" on the right

2. **Hero Section**
   - Full-screen background image (tropical leaves/nature)
   - Large serif heading: "traduzimos mulheres que marcam o mundo."
   - Subtitle text about the boutique branding agency
   - Dark overlay for text readability

3. **Value Proposition Banner**
   - Beige/cream background
   - Centered italic text about personal branding as an investment

4. **Quote Section**
   - Large quote: "em um mundo com 8 bilhões, só você é você..."
   - Decorative typography treatment

5. **Three Differentiators**
   - Three bullet points with bold leads explaining what makes them unique (personalized journey, sensitive perspective, balance as a tool)

6. **Services Section ("SERVIÇOS")**
   - Section title with decorative subtitle
   - Three service cards in a row, each with:
     - Image
     - Service name (Consultoria Voo, Mentoria SALto, Cursos)
     - Brief description
     - Click/link to detail

7. **FAQ Section (Accordion)**
   - "Perguntas Frequentes" heading
   - 4 expandable questions using the Accordion component
   - WhatsApp contact link at the bottom

8. **CTA Section**
   - "Pronta para SALtar?" heading
   - Large call-to-action button

9. **Footer**
   - Minimal footer with contact info and social links

### Pages
- Single-page layout with smooth scroll anchors
- All content on the Index page

### Technical Notes
- Use placeholder images from Unsplash (tropical leaves, nature, women professionals)
- Google Fonts: Playfair Display + Inter/Lato
- Fully responsive (mobile-first)
- Smooth scroll behavior
- FAQ uses existing Accordion component
